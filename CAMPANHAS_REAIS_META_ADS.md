# 🎯 Como Ver Campanhas REAIS do Meta Ads

## ✅ Sistema Atualizado!

Agora o sistema pode mostrar **campanhas reais** do Meta Ads, não apenas dados de teste!

## 🔄 Como Funciona:

### 1. **Dados de Teste** (atual)
- Sistema mostra campanhas fictícias para testar a interface
- Útil para desenvolvimento e demonstração

### 2. **Campanhas Reais** (novo!)
- Sistema busca campanhas diretamente do Meta Ads
- Dados reais de performance, gastos, impressões, etc.

## 🚀 Como Ativar Campanhas Reais:

### Passo 1: Conectar Meta Ads
1. Vá para **Dashboard > Clientes**
2. Selecione "Coan Consultoria"
3. Na seção "Meta Ads", clique **"Conectar Meta Ads"**
4. Faça login no Meta e autorize o acesso
5. Selecione as contas de anúncios desejadas

### Passo 2: Sincronizar Campanhas
1. Volte para **Dashboard > Campanhas**
2. Selecione "Coan Consultoria"
3. Clique no botão **🔄** (sincronizar) ao lado de "Carregar Campanhas"
4. Aguarde a sincronização das campanhas reais

### Passo 3: Ver Campanhas Reais
1. Clique **"Carregar Campanhas"**
2. Agora você verá as campanhas REAIS do Meta Ads!

## 🔍 Como Identificar:

### Campanhas de Teste:
- ✅ Mensagem: "Dados de teste - API funcionando perfeitamente!"
- ✅ Nomes: "Campanha de Teste - ..."
- ✅ Conta: "Conta Meta Teste"

### Campanhas Reais:
- 🎯 Mensagem: "Campanhas reais carregadas com sucesso!"
- 🎯 Nomes: Nomes reais das suas campanhas
- 🎯 Conta: Nome real da conta Meta
- 🎯 Dados: Gastos, impressões, cliques reais

## 🛠️ APIs Criadas:

1. **`/api/meta/sync-campaigns`** - Sincroniza campanhas do Meta
2. **`/api/dashboard/campaigns-fixed`** - Busca campanhas (reais ou teste)

## 📊 Fluxo Completo:

```
1. Conectar Meta Ads → 2. Sincronizar → 3. Ver Campanhas Reais
     (uma vez)           (quando quiser)      (sempre atualizadas)
```
