Sistema de Gerenciamento de Assinaturas

📋 Instruções para Teste

Siga os passos abaixo para testar o modal de ajuste manual:

1️⃣ Verificar Servidor

Certifique-se de que o servidor Next.js está rodando:

npm run dev
⏳ Verificando servidor...

2️⃣ Acessar Página

Clique no botão abaixo para abrir a página de gerenciamento:

🚀 Abrir Subscription Management

3️⃣ Executar Teste Automático

Na página que abrir, pressione F12 para abrir o DevTools, vá na aba Console e execute:

// Carregando script de teste...

4️⃣ Teste Manual

Alternativamente, teste manualmente:

  • Procure por organizações na tabela
  • Clique no botão "Ajustar" de qualquer organização
  • Verifique se o modal abre com o formulário completo
  • Confirme se há campos: Tipo de Ajuste, Motivo, Valor

🔧 Solução de Problemas

Se o modal ainda mostrar "será implementada em breve":

Remove-Item -Recurse -Force .next
npm run dev

Ou pressione Ctrl+F5 para forçar refresh do cache.
